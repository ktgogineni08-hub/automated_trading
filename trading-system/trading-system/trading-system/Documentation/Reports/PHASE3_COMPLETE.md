```bash

# Access services
```

### 2. Run Tests

```bash
# Run all tests
pytest tests/test_core_modules.py -v

# Run specific test
pytest tests/test_core_modules.py::TestMLSignalScorer -v

# With coverage
pytest tests/test_core_modules.py --cov=core
```

### 3. Deploy to Production

```bash
# Deploy to Kubernetes
kubectl apply -f k8s/namespace.yml
kubectl apply -f k8s/secrets.yml  # After adding secrets!
kubectl apply -f k8s/configmap.yml
kubectl apply -f k8s/storage.yml
kubectl apply -f k8s/deployment.yml

# Verify
kubectl get pods -n trading
kubectl get svc -n trading
```

### 4. Use Individual Components
